enum core::partition_codes

Contains all possible Result Codes of any operation on a partition.

Mostly used by validators.

Version
1.0.0.0
Enumerator
OK 

Ok.

data_invalid 

Invalid Data.

address_underflow 

Address Underflow.

address_overflow 

Address Overflow.

partition_mounted 

Partition is mounted.

partition_not_mounted 

Partition is not mountet.

index_out_of_bounds 

Array has been accessed with an illegal index.

no_definition 

Definition is missing.

file_not_exist 

File does not exist.

file_open_failure 

Can't open file.

mapping_failure 

Mapping failed.

unknown 

Unknown result.
