static virtual_disk_file* core::disk::load_master_boot_record ( const char *  disk_path)
static

This static method loads a master boot record from a virtual disk file.

This method opens the virtual disk file and maps the content into memory. After the mapping the virtual disk file is closed again and the mapped data is inspected. Finally the inspected data is unmapped from the memory.
