◆ print()

static int utils::menu::print ( const char *  entries[],
int  amount_entries,
const char *  menu_header,
menu_settings &  settings 
)
inlinestatic

Prints a menu on the terminal.

This method prints every menu entry on the terminal and allows the selection with specified keys. The menu is customizable via a settings argument. The result value correspond to the provided entries, e.g. result 0 <=> entries[0] A known bug is that the input stream is filled after some operation, therefore setting the settings entry clear_cache is recommended

Parameters
entriesMenu entries which should be printed
amount_entriesAmount of the entries
menu_headerHeader of the menu
settingsSettings object
Returns
int Selected 0 based index
